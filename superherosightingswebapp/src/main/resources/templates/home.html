Introduction

This is a Superhero Sightings Web Application that manages Superheroes, Superpowers, Sightings, Locations and Organizations. This application performs all the CRUD operations on these entities and also lists the most recent Sightings for Superheroes. This application comprises of many database relationships such as the one-to-many relationship between Superpower and Superhero, and many-to-many relationship between Superhero and Organization.
